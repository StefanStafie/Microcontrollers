Before running:
Make sure you have the right proteus version and the xc8 compiler.
Change the path to lcd.h to the right one (line 19).


Extra info:
Run and press all buttons.
Make sure the buttons stay pressed (use the red circle with arrows). 
The buttons are labeled.
The button is working if it has red square.
Check osciloscope while pressing buttons.
